Methodology

Adapted from Sehgal, Tronieri, Ungar & Guntuku (2026). "Self-Reported Side Effects of Semaglutide and Tirzepatide in Online Communities." arXiv:2603.12341 / medRxiv DOI: 10.64898/2026.03.12.26348253

Pipeline

  • Data Collection: 2,981 Reddit posts scraped from r/Peptides, r/Nootropics, r/bpc_157 via Arctic Shift archive API
  • LLM Classification: Each post processed through qwen3.5:9b running locally via Ollama. The model classified self-use, identified peptides, and extracted effects with valence (positive/negative/neutral) and confidence labels
  • Effect Normalization: 25 canonical effect groups mapped from raw LLM output (e.g., "anxiety relief", "reduced anxiety", "less anxiety" → anxiety_positive)
  • Literature Review: Published research gathered via web search, with citations verified through Crossref/OpenAlex. Effects compared as concordant, discordant, or novel

Technical Details

  • Encoding: 2,979/2,981 posts successfully parsed (99.93%)
  • Runtime: ~4.2 hours on local hardware (qwen3.5:9b via Ollama)
  • API cost: $0 (entire pipeline runs locally)
  • 1,777 posts classified as self-reports (59.6%)
  • 3,657 individual effects extracted with valence tags
  • Checkpointing every 10 posts for crash recovery

Key Findings

  • BPC-157 anhedonia/anxiety signal: Community reports show net adverse neuropsychiatric effects that contradict animal data showing anxiolytic properties. This is a genuine novel pharmacovigilance signal worth investigating.
  • Selank anxiety relief confirmed: 9.7% positive anxiety reports strongly concordant with Russian clinical trial data showing benzodiazepine-comparable efficacy.
  • Semax cognitive effects confirmed: 5.7% positive focus reports concordant with published nootropic evidence. However, 3.9% brain fog reports are discordant.
  • Hair loss from Semax: 1.6% report hair loss with zero published data. Novel signal with theoretical pathway via melanocortin system.

Limitations

  • No sentiment disambiguation at sentence level (post-level valence only)
  • Small model (9B) for extraction; larger models would improve accuracy
  • Posts only, no comments (Arctic Shift comment API was unavailable)
  • No dose extraction or dose-response analysis
  • Selection bias: users with notable effects post more
  • Source quality unknown: unregulated peptide sources may be impure
  • Russian clinical literature largely inaccessible for direct verification
