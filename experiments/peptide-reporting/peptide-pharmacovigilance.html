BPC-157 — The Healing Peptide

1,107 self-reports. Top positive: pain relief (3.2%), healing acceleration, weight loss. Top adverse: fatigue (2.2%), anhedonia (2.0%), insomnia (1.5%). Notable neuropsychiatric signal — anhedonia and anxiety reports are underappreciated in the "healing peptide" narrative.

Selank — The Anxiolytic Peptide

411 self-reports. Strongest signal: anxiety relief (10.5%) — highest positive-effect rate of any peptide-effect pair. But also anxiety adverse (5.6%) — suggesting it works for some, worsens for others. Focus improvement (2.9%) also notable.

Semax — The Cognitive Enhancer

470 self-reports. Top positive: focus improvement (5.1%), anxiety relief (4.7%), mental clarity (2.3%). Top adverse: brain fog (4.0%) — paradoxical for a nootropic. Hair loss (1.3%) is a distinct signal not seen with the others.

Methodology

Adapted from: Sehgal, Tronieri, Ungar & Guntuku (2026). "Self-Reported Side Effects of Semaglutide and Tirzepatide in Online Communities." arXiv:2603.12341 / medRxiv DOI: 10.64898/2026.03.12.26348253

Pipeline

  • Scraping: 2,981 Reddit posts from r/Peptides, r/Nootropics, r/bpc_157 via Arctic Shift API. Queries: BPC-157, BPC157, selank, semax.
  • LLM Classification: Each post processed by qwen3.5:9b (local, via Ollama) to identify: (1) self-report status, (2) peptides mentioned, (3) effects with valence (positive/negative/neutral), confidence, and MedDRA SOC category.
  • Effect Normalization: ~2,700 raw LLM effect strings consolidated into canonical names via mapping table (e.g., "anxiety reduction", "reduced anxiety", "anti-anxiety" → "Anxiety relief").
  • Total processing time: ~4.2 hours on local hardware (qwen3.5:9b, ~5.5 sec/post).

Key Improvement over v1

The v1 regex-only approach could not distinguish direction — "anxiety" meant both "caused anxiety" and "treated anxiety." The LLM encoder separates these, revealing that e.g. Selank has a 10.5% anxiety relief rate but also 5.6% anxiety worsening rate.

Limitations

  • Selection bias: Reddit users with notable effects post more.
  • No causation: Co-occurrence with peptide use ≠ causation.
  • LLM accuracy: 9B model may misclassify nuanced language. Future: validate sample against human coders.
  • No dose-response: Doses mentioned but not systematically extracted.
  • Posts only: Comments (often richer data) not scraped (API limitation).

Reproducibility

Full pipeline is reproducible: data collected via Arctic Shift public API, encoded by local open-weight model (qwen3.5:9b). All intermediate data (raw posts, LLM outputs as JSONL, normalized dashboard data) preserved. No proprietary API calls — entire pipeline runs on consumer hardware.

Future Directions

  • Dose extraction and dose-response curves
  • Temporal analysis (user posting history over weeks/months)
  • Cross-platform: forums, Discord, Telegram peptide groups
  • Fine-tuned classifier for higher accuracy
  • Comparison with published clinical/preclinical data where available
