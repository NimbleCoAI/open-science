🔬 Methods

Data Source

  • Xeno-Canto API v3 — all recordings geotagged to New Zealand
  • 2,244 bird recordings across 175 species (excluding soundscapes & unidentified)
  • Quality ratings: A (902), B (948), C (316), D (48), E (9)

Syllable Analysis Pipeline (replicating Hill & Ji 2014)

  • Audio: Resampled to 44.1kHz mono (matching Hill's Marantz PMD620)
  • Spectrogram: DFT=256, Hann window, 50% overlap (Hill's Raven Pro settings)
  • Song detection: Band-limited energy thresholding (0.5–10kHz), median-filtered
  • Syllable parcellation: ≥20ms pause criterion with dynamic energy threshold
  • Feature extraction: f0 (YIN), MFCCs (13 coefficients), spectral centroid/bandwidth/rolloff/flatness, ZCR, FM rate
  • Classification: Rule-based following Hill's 6 categories (low-freq <2kHz, high-freq ≥5kHz, harmonic, trill, RMNR, harsh)
