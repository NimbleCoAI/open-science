🔬 Hill's Methods & Proposed Pipeline

Hill & Ji (2014) — Original Analysis

  • Equipment: Sennheiser ME67 shotgun mic + Marantz PMD620 recorder (44.1 kHz)
  • Software: Raven Pro 1.4 Beta (Cornell Lab)
  • Settings: DFT=256, Hann window, 2.9ms, 50% frame overlap, bandwidth=3dB
  • Manual syllable categorisation by visual/aural inspection of spectrograms
  • 6 syllable types: harmonic, RMNR, harsh, trill, high-freq (≥5kHz), low-freq (<2kHz)

Priyadarshani et al. (2018) — Review of Automated Methods

  • 4-stage pipeline: preprocessing → detection → feature extraction → classification
  • Noise reduction: bandpass filtering, spectral subtraction, wavelet denoising
  • Detection: energy thresholding, spectrogram cross-correlation, HMMs
  • Features: MFCCs (most common), mel spectrograms, spectral features
  • Classification: Random Forest (up to 96%), SVMs, neural networks
  • AviaNZ: open-source tool by same group, designed for NZ birds

Proposed Automated Pipeline for This Project

  • Step 1: Download high-quality (A/B rated) foreground recordings from XC via API
  • Step 2: Noise reduction using wavelet denoising (Priyadarshani et al. 2016)
  • Step 3: Song detection via energy thresholding on band-limited spectrograms
  • Step 4: Syllable segmentation using ≥20ms pause criterion + spectral clustering
  • Step 5: Feature extraction (MFCCs, spectral features) per syllable
  • Step 6: Unsupervised syllable clustering to replicate Hill's 6-type categorisation
  • Step 7: Validate against Hill's published proportions
